`Python으로 챗봇 만들어보기`_ 실습 예제

`BotHub.Studio`_ 를 사용해서 개봉 영화를 확인하고 근처 상영관의 상영시간표를 확인하는 흐름을 구현함.

Telegram과 Facebook Messenger를 기준으로 테스트되었음.

예제에 사용한 롯데시네마는 해외 서버의 접속을 차단하고 있어서 그냥 사용할 수는 없고, 국내의 릴레이 서버를 사용해야 함.
